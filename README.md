# 🌱 GreenLeaf Plant Shop

## 📝 Description

This is a plant shopping web application built using React and Redux. Users can browse different plants, add them to the cart, and manage their shopping cart efficiently.

---

## 🚀 Features

* View different categories of plants (Indoor, Outdoor, Succulents)
* Add plants to cart
* Increase and decrease quantity
* Remove items from cart
* View total items and total price

---

## 🛠️ Technologies Used

* React.js
* Redux Toolkit
* React Router
* CSS / Bootstrap
